# HTML Notes

## HTML
HTML is a markup language. It was developed by Sir Tim Berners Lee in 1991. It stands for **Hyper Text Markup Language**.

It is used to create the structure of a webpage. A collection of static webpages is called a website.

---

## HTML Versions

- **HTML4** – 84 tags(1997)
- **XHTML**
- **HTML5** – 120+ tags(2014)

---

## Important Terms

- **DNS** – Domain Name System (1989)
- **WWW** – 1990

---

## Tag

Everything written in angular brackets is called a **tag**. It is the smallest unit of HTML.

There are two types of tags:

### 1. Container Tag

Those tags which have an opening and a closing tag are called **container tags**. They can contain something.

**Examples:**

```html
<html></html>
<head></head>
<p></p>
```

### 2. Self Closing Tag / Empty Tag

These tags close themselves.

**Examples:**

```html
<br/>
<link/>
<img/>
<hr/>
```

Example:

```html
<Akash>
```

# HTML Body Tags Notes

## Tags of Body

### 1. Marquee Tag
- It is a container tag.
- Used to scroll the text.

### 2. Heading Tag
- Used to make headings from **`<h1>`** to **`<h6>`**.

### 3. Paragraph Tag

```html
<p></p>
```

- This tag is used to create paragraphs.

### 4. Italic Tag

```html
<i></i>
```

- This tag is used for italics.

### 5. Underline Tag

```html
<u></u>
```

- This tag is used to underline text.

### 6. Bold Tag

```html
<b></b>
```

- This tag is used to make text bold.

---

# Block Level Elements

Block level elements cover **100% width** of their parent tag.

**Examples:**
- `<head>`
- `<body>`
- `<h1>` to `<h6>`

---

# Inline Elements

Inline elements cover only the required width for their content.

**Examples:**
- `<i>`
- `<b>`
- `<u>`

---

# Semantic Tags

Semantic tags are those whose working or functionality is self-defined by their name.

These were introduced in **HTML5**.

**Examples:**

```html
<video></video>
<audio></audio>
<strong></strong>
<mark></mark>
```

---

# Image Tag

```html
<img />
```

It is used to embed images in web pages.

It is **not a semantic tag** because it was introduced before HTML5.

## Attributes of `<img>` Tag

- `src`
- `height`
- `width`
- `alt`
- `title`
